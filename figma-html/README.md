## Что создано
В папке лежит комплект для импорта в Figma через плагин `html.to.design`:

- `customer.html` — страница Figma `1) Заказчик`
- `manager.html` — страница Figma `2) Менеджер`
- `support-admin.html` — страница Figma `3) Техподдержка+Админ`
- `ui.css` — общий стиль
- `design-tokens.json` — токены (для контроля палитры/типографики)

Также добавлены самодостаточные файлы **без внешнего CSS** (для импорта “по файлу” / через конвертер без URL):
- `customer.inline.html`
- `manager.inline.html`
- `support-admin.inline.html`

Каждый HTML содержит несколько `artboard` 1440×900 — это будущие фреймы/экраны.

## Как импортировать в Figma (лимит 3 страницы) — без хостинга/серверов
1. Откройте Figma → новый файл (Design).
2. Создайте 3 страницы:
   - `1) Заказчик`
   - `2) Менеджер`
   - `3) Техподдержка+Админ`
3. Установите плагин `html.to.design`:
   - `Resources` (Shift+I) → `Plugins` → поиск `html.to.design` → `Install`
4. Импорт по страницам (по одному файлу на страницу):
   - страница `1) Заказчик` → Run `html.to.design` → импортируйте `customer.inline.html`
   - страница `2) Менеджер` → импортируйте `manager.inline.html`
   - страница `3) Техподдержка+Админ` → импортируйте `support-admin.inline.html`

## Если стили не подтянулись
Используйте `*.inline.html` (там CSS уже встроен).
